# 🏦 대출 상환 비교 플래너

한국 대출 환경에 맞는 상환 방식을 한눈에 비교할 수 있는 웹 앱입니다.

## ✨ 주요 기능

### 📊 4가지 상환 방식 비교
- **원리금균등상환** — 매달 동일한 금액 납부
- **원금균등상환** — 매달 동일한 원금 + 잔여 이자
- **만기일시상환** — 이자만 납부 후 만기에 원금 상환
- **거치식상환** — 거치 기간 후 원리금균등 전환

### 📈 시각적 비교
- 요약 카드 (총이자, 월 상환액, 최대 상환액)
- 월별 상환액 추이 차트
- 대출 잔액 변화 차트
- 이자 비중 변화 차트
- 총이자 비교 바 차트

### ⚡ 중도상환 시뮬레이션
- 특정 시점에 추가 상환 시 이자 절감 효과 확인

### 🎨 디자인
- 다크/라이트 모드 지원
- 모바일 반응형 레이아웃
- 글래스모피즘 카드 디자인

## 📖 사용 방법

1. `index.html`을 브라우저에서 열기
2. 대출 원금, 이자율, 기간 입력
3. 비교할 상환 방식 선택
4. **상환 비교 계산하기** 클릭
5. 결과 확인 및 차트 탭 전환

## 🛠 기술 스택

- HTML / CSS / JavaScript (순수 웹)
- [Chart.js](https://www.chartjs.org/) (CDN)
- [Noto Sans KR](https://fonts.google.com/noto/specimen/Noto+Sans+KR) (Google Fonts)
