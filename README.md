# 🏦 내게 딱 맞는 대출 찾기: 대출 상환 비교 플래너

**현재 버전**: `v0.9.5` (Release Candidate)  
**마지막 업데이트**: 2026-02-20

"어떤 상환 방식이 나에게 가장 유리할까?" 고민되시나요?  
복잡한 금융 용어와 계산 대신, 직관적인 차트와 리포트로 한눈에 비교해 보세요.

---

## 🌟 왜 이 플래너를 사용해야 하나요?

*   **돈 아끼는 선택**: 어떤 방식이 이자를 가장 적게 내는지 정확한 수치로 보여줍니다.
*   **현금 흐름 파악**: 매달 내 지갑에서 나가는 돈의 변화를 시각적인 차트로 확인하세요.
*   **중도상환 시뮬레이션**: "갑자기 돈이 생겨서 미리 갚으면 얼마나 이득일까?"를 즉시 계산해 보세요.
*   **스마트한 입력**: 억 단위 자동 변환 기능이 있어 큰 금액도 실수 없이 입력할 수 있습니다.

---

## 📊 주요 기능 안내

### 1. 4가지 핵심 상환 방식 자동 비교
*   **원리금균등**: 매달 같은 금액을 내고 싶을 때 (예산 계획에 유리!)
*   **원금균등**: 초기에 더 갚더라도 전체 이자를 줄이고 싶을 때 (가장 알뜰한 선택!)
*   **만기일시**: 당장은 이자만 내며 현금을 확보하고 싶을 때 (투자금 확보에 유리!)
*   **거치식**: 일정 기간 이자만 내다가 나중에 나눠 갚고 싶을 때 (초기 부담 완화!)

### 2. 스마트 분석 리포트
*   **추천 뱃지**: '최저 이자', '현금 흐름 확보' 등 사용자 상황에 맞는 최적의 옵션을 뱃지로 알려드립니다.
*   **이자 비중 차트**: 시간이 흐를수록 내가 내는 돈에서 이자가 얼마나 줄어드는지 확인하세요.
*   **비교 데이터 수출**: 분석한 결과를 🖼️ **이미지**로 저장하거나 **엑셀(CSV)** 파일로 소장할 수 있습니다.

---

## � 사용 방법 (3분 완성)

1.  **대출 정보 입력**: 대출 원금, 연 이자율, 대출 기간을 입력합니다. (숫자를 치면 '3억 5,000만원'처럼 한글로 친절히 알려드려요!)
2.  **방식 선택**: 비교해보고 싶은 상환 방식들을 체크합니다.
3.  **계산하기**: **[상환 비교 계산하기]** 버튼을 클릭합니다.
4.  **결과 확인**: 하단에 나타나는 요약 카드와 차트를 보며 나에게 유리한 방식을 고릅니다.
5.  **보관하기**: '결과 이미지 저장' 버튼으로 스마트폰이나 PC에 결과를 저장해 두세요.

---

## 🌙 테마 및 모바일 지원

*   **다크 모드 지원**: 낮이나 밤이나 눈이 편안한 테마로 바꿀 수 있습니다.
*   **모바일 최적화**: 스마트폰에서도 앱처럼 깔끔하게 작동합니다.
*   **개인정보 보호**: 이 도구는 입력한 데이터를 서버로 전송하지 않습니다. 모든 계산은 내 브라우저에서 안전하게 이루어집니다.
